# AI Chatbot

An AI-powered chatbot built using React.js, Node.js, Express.js, and Groq API with real-time streaming AI responses.

## Features

- Real-time AI responses
- Streaming chatbot output
- Responsive React UI
- Backend API integration
- Groq LLM integration
- Clean conversational interface

## Tech Stack

- React.js
- Node.js
- Express.js
- JavaScript
- Groq API
- REST APIs

## Backend Deployment

Deployed on Render.

## Frontend Deployment

Deployed on Vercel.
